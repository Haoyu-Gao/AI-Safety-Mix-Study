# ICSE-submission-47

## AI Safety in the Eyes of the Downstream Developer: A First Look at Concerns, Practices, and Challenges

This is the replication package for submission 47 for ICSE'26.

### Folder Walkthrough
- `intervew` folder contains our interview protocol named `Interview Protocol.docx`, and our thematic analysis results `Coding Summary.xlsx`.
- `survey` folder contains our survey screenshot named `Survey Screenshot.pdf` and survey results converted in to 4/5 point Likert scores in file named `Survey Results.xlsx`.
- `incident database` folder contains our analysis for the AI incident database (in `AI Incident DB Analysis.xlsx`). Each row contains the description for an AI safety failure in real world scenarios.

The original coding process was conducted in Nvivo. However, due to ethical approval restrictions, we cannot release the raw transcript data. Instead, the Coding Summary.xlsx provides a detailed analysis, including key points extracted from the raw data for better illustration.
